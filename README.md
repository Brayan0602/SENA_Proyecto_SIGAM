# SIGAM

Sistema de Gestión Académica desarrollado con:

- React
- Vite
- Node.js
- Express
- MongoDB Atlas

## Funcionalidades

- Inicio de sesión
- Gestión de estudiantes
- Crear estudiantes
- Consultar estudiantes
- Actualizar estudiantes
- Eliminar estudiantes

## Instalación

Frontend:

pnpm install
pnpm dev

Backend:

npm install
node src/server.js
